1. Git 활용 도구 소개 및 Git을 활용한 소스코드 관리

   - GitHub 활용법(프로젝트 관리, 오픈소스 프로젝트, 위키 작성)

   - Flask 기반의 기초 챗봇 서버 구축
   - PaaS 서비스를 활용한 서비스 배포

   - 메세징 기초, Webhook를 활용한 응답형 챗봇 제작
